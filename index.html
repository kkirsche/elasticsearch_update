Elasticsearch Update

by Kevin Kirsche

Overview

Elasticsearch Update allows you to update individual nodes in an Elasticsearch cluster using either a .deb package or a .rpm package.

Installation

Installing the gem is easy, simple execute gem install elasticsearch_update to install the gem. You can then update your local node by running the command elasticsearch_update. You will then begin the prompts to gather the information necessary to update your local node.

For development, clone the project and then navigate to the folder. In the folder, execute bundle exec bin/elasticsearch_update and the gem will walk you through the upgrade process.

Versioning

For transparency into my release cycle and in striving to maintain backward compatibility, Elasticsearch Update is maintained under the Semantic Versioning guidelines. Sometimes I screw up, but I'll adhere to those rules whenever possible.
